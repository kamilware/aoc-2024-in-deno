# aoc-2024-in-deno

Solutions for Advent of Code 2024 in deno

# Input

Input files are not included as per AOC terms. Place them inside "<root>/input"
(same lvl as "test_input") folder

- 1.txt for day 1
- 2.txt for day 2
- 3.txt for day 3
- ...

# Running
To get output run the shell script:
- for day 1
```bash
sh d1.sh
``` 
-  for day 2
```bash
sh d2.sh
```
- for day 3
```bash
sh d3.sh
``` 
- ...
